ℹ About TaskFlow

TaskFlow is a personal task manager with:

🎓 Collage Tasks — spaced repetition scheduling (+0,+1,+3,+6,+10,+15,+21,+42,+64 days)
🔁 Repeated Work — auto-added to every day of the year
📅 Calendar — visual task presence indicators
⏱ Timers — stopwatch, countdown, fullscreen clock
📊 Stats — charts powered by Chart.js
📂 AI/ML Files — password-protected file vault (base64 in localStorage)
💾 Auto-save — notes auto-save on blur; save button for everything
📤 Export/Import — full JSON backup system

All data stored in browser localStorage.
